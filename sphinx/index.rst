.. title:: sgbackup API

Descritpion
-----------

Sgbackup is a savegame backup tool written in Python. It has a commandline and a graphical 
interface using *Gtk4*.

The minimum Python version is **3.11**.

It has\ *Gtk4*\ and\ *GObject Introspection*\ as a dependency.
